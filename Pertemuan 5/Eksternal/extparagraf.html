Cascading Style Sheets (CSS) adalah bahasa desain web (style sheet language) yang mengontrol format tampilan dari sebuah halaman website yang ditulis menggunakan markup language. CSS biasanya digunakan untuk mendesain halaman HTML dan XHTML, tetapi kini bisa diaplikasikan ke dokumen XML lainnya seperti SVG, XUL, bahkan Android.
